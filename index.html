#-------------------------------------------------------------------------------

git init
git add README.md | Добавить в индекс файл README.md
git commit -m "first commit" | Создать коммит с комметарием "first commit"
git remote add origin https://github.com/classtype/project.git
git push -u origin master

git status | Показать статус.
git clone https://github.com/classtype/project.git ~/project | Клонировать репозиторий в директорию project
git remote add origin https://github.com/classtype/project.git | Привязать удаленный репозиторий "origin" к локальному.
git remote rm origin | Отвязать удаленный репозиторий "origin" от локального.
git push -u origin master | Связать удаленный репозиторий "origin" с локальной веткой "master".
git pull | Скачать изменения с удаленного репозитория.

#-------------------------------------------------------------------------------
# Буфер обмена
#-------------------------------------------------------------------------------

git stash | Сохранить изменения в бумер обмена.
git stash pop | Выгрузить изменения сделаные предыдущей командой.

#-------------------------------------------------------------------------------
# Рабочая директория
#-------------------------------------------------------------------------------

git checkout @ . | Сбросить индекс и откатить все изменения в рабочей директории.
git checkout . | Откатить все изменения до версии в индекс, если индекс пуст, то до версии коммит.
git checkout 6c2e81b . | Взять все изменения из коммит "6c2e81b" и добавить их в индекс и рабочую директорию.

#-------------------------------------------------------------------------------
# Индекс
#-------------------------------------------------------------------------------

git add index.html | Добавить файл "index.html".
git reset index.html | Сбросить файл "index.html".
git reset | Сбросить все изменения.
git add . | Добавить все изменения из текущей директории.
git add --all | Добавить все изменения из всех поддиректорий.
git add -A | Тоже самое, только сокращенный вариант.

#-------------------------------------------------------------------------------
# Переместить указатель в текущей ветке на другой коммит
#-------------------------------------------------------------------------------

git reset --hard 6c2e81b | С удалением всех изменений в рабочей директории и в индекс.
git reset --soft 6c2e81b | С сохранением текущих изменений в рабочей директории и в индекс.
git reset 6c2e81b | С сохранением текущих изменений только в рабочей директории.
git reset --mixed 6c2e81b | Тоже самое, так как --mixed это флаг по умолчанию.

#-------------------------------------------------------------------------------
# Коммит
#-------------------------------------------------------------------------------

git commit --amend -m "New comment" | Перезаписать последний коммит с комментарием "New comment".
git commit --amend --no-edit | Тоже самое, но без изменения комментария.
git commit -a -m "Comment" | Сделать коммит в обход git add, но только для файлов которые уже отслеживаються.

#-------------------------------------------------------------------------------
# Список всех коммит
#-------------------------------------------------------------------------------

git log | Просмотра списка всех коммит.
git log --oneline | Каждый коммит показывается в одну строчку.

#-------------------------------------------------------------------------------
# Изменения в коммит
#-------------------------------------------------------------------------------

git show | Показать изменения в последнем коммит.
git show --name-only | Тоже самое, но показать только имена измененных файлов.
git show 6c2e81b | Просмотреть изменения по конкретному коммит.
git show master | Просмотреть изменения по последнему коммит в ветке master.
